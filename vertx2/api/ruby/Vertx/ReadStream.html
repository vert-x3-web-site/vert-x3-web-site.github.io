Overview

A mixin module which represents a stream of data that can be read from. Any class that mixes in this module can be used by a Pump to pump data from a ReadStream to it.

def data_handler(&hndlr)
  @j_del.dataHandler(Proc.new { |j_buff|
    hndlr.call(Buffer.new(j_buff))
  })
  self
end
