Community

Without the community we are nothing.

Vert.x is an independent project and has been created by contributions from many different individuals and organisations.

We have a thriving user and developer community and many exciting new developments to come, so why not be a part of it?

Get Help

User Group

A Google Group for users. Post a message if you need help or to report problems.

Stack Overflow

Mark your questions with the vert.x tag. We do our best to answer questions there but keeping up with the User Group is our priority.

Gitter Channel

Come and chat with other users and module maintainers.

Bug reports

If you're not sure which module is the cause of your problem, file a new entry on vert-x3/issues. Otherwise you can file directly to eclipse/vert.x or any of the vert-x3 modules.

Wiki

The Vert.x wiki contains useful information such as a How to contribute guide, the Roadmap and Project Information.

Developer Group

A Google Group for Vert.x core developers and contributors.

Vert.x awesome

A curated list of awesome frameworks, libraries or other components for use with Vert.x. Feel free to add yours by submitting a pull request.
