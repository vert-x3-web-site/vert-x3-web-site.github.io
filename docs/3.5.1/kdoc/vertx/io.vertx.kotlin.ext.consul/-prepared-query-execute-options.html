fun PreparedQueryExecuteOptions(limit: Int? = null, near: String? = null): PreparedQueryExecuteOptions

A function providing a DSL for building io.vertx.ext.consul.PreparedQueryExecuteOptions objects.

Options used to execute prepared query

Parameters

limit - Set the size of the list to the given number of nodes. This is applied after any sorting or shuffling.

near - Set node name for sorting the list in ascending order based on the estimated round trip time from that node. Passing _agent will use the agent's node for the sort. If this is not present, the default behavior will shuffle the nodes randomly each time the query is executed.
