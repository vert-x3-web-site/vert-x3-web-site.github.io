resolveA

open fun resolveA(name: String, handler: Handler<AsyncResult<MutableList<String>>>): DnsClient

Try to resolve all A (ipv4) records for the given name.

Parameters

name - the name to resolve

handler - the io.vertx.reactivex.core.Handler to notify with the io.vertx.reactivex.core.AsyncResult. The handler will get notified with a java.util.List that contains all the resolved java.net.Inet4Addresses. If none was found an empty java.util.List will be used. If an error accours it will get failed.

Return
a reference to this, so the API can be used fluently
