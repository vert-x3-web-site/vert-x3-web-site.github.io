        • DnsClient lookup(java.lang.String name, io.vertx.core.Handler<io.vertx.core.AsyncResult<java.lang.String>> handler)

          Try to lookup the A (ipv4) or AAAA (ipv6) record for the given name. The first found will be used.

          Returns:
          a reference to this, so the API can be used fluently
          Parameters:
          name - the name to resolve
          handler - the io.vertx.groovy.core.Handler to notify with the io.vertx.groovy.core.AsyncResult. The handler will get notified with the resolved address if a record was found. If non was found it will get notifed with null. If an error accours it will get failed.
