[Groovy] Class ShellServer

    • io.vertx.groovy.ext.shell.ShellServer

    • The shell server.

      A shell server is associated with a collection of : the ShellServer.registerTermServer method registers a term server. Term servers life cycle are managed by this server.

      When a receives an incoming connection, a instance is created and associated with this connection.

      The ShellServer.createShell method can be used to create instance for testing purposes.
