JDBCAuthOptions

fun JDBCAuthOptions(authenticationQuery: String? = null, config: JsonObject? = null, datasourceName: String? = null, permissionsQuery: String? = null, rolesPrefix: String? = null, rolesQuery: String? = null, shared: Boolean? = null): JDBCAuthOptions

A function providing a DSL for building io.vertx.ext.auth.jdbc.JDBCAuthOptions objects.

Options configuring JDBC authentication.

Parameters

authenticationQuery - Set the authentication query to use. Use this if you want to override the default authentication query.

config - The configuration of the JDBC client: refer to the Vert.x JDBC Client configuration.

datasourceName - Set the data source name to use, only use in shared mode.

permissionsQuery - Set the permissions query to use. Use this if you want to override the default permissions query.

rolesPrefix - Set the role prefix to distinguish from permissions when checking for isPermitted requests.

rolesQuery - Set the roles query to use. Use this if you want to override the default roles query.

shared - Set whether the JDBC client is shared or non shared.
